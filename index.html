Welcome to Falling Blocks.

This is a Tetris clone, created by @Thinic, because there are not many clones that allow two players to play with the same keyboard. You can download the latest file here. The game requires Java 7 or higher as of 6 December 2015.

How to Play

Solo

Left - Left
Right - Right
Down - Down
Instant Down - Space
Rotate CW - Up, X
Rotate CCW - Z

Duel

Player 1

Left - A
Right - D
Down - S
Instant Down - Q
Rotate CW - W, X
Rotate CCW - Z

Player 2

Left - Left
Right - Right
Down - Down
Instant Down - End
Rotate CW - Up, Page Down
Rotate CCW - Delete
